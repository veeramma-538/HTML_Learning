## 1️⃣ Introduction to HTML

- **HTML** = *HyperText Markup Language*  
- It defines the **structure** of a webpage  
- Uses **tags** enclosed in angle brackets (`< >`)  
- Example:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>
